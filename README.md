# Proyecto 3 — MTF / IMTF

Implementación de los algoritmos **Move-to-Front (MTF)** e **Improved Move-to-Front (IMTF)** para el análisis de costos de acceso en listas auto-organizadas. Se calculan y comparan costos sobre distintas secuencias de solicitudes, incluyendo el mejor y peor caso, como parte del estudio del análisis amortizado y competitivo de algoritmos online.

## Requisitos

- Python 3.10+

## Instalación

```bash
git https://github.com/GenserDev/Proyecto-3-Algoritmos.git
cd Proyecto-3-Algoritmos
```

## Uso

```bash
python main.py
```

Se abrirá un menú interactivo con las 6 tareas del proyecto. Selecciona el número de la tarea que deseas ejecutar, o `7` para correr todas.
